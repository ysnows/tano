## What's working end-to-end on iOS
1. `tano create my-app` scaffolds a project
2. `tano dev` starts Bun server + opens simulator
3. `tano build ios` bundles server + web
4. App server runs via TanoJSC (Bun.serve compatible)
5. WebView renders UI with window.Tano.invoke() bridge
6. 11 native plugins accessible from WebView
7. `tano doctor` validates environment
