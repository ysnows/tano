---
title: 'tano doctor'
description: 'Check your development environment'
---

## Usage

```bash
tano doctor
```

## What it checks

- Bun installation and version
- Xcode installation and command-line tools
- Android Studio and SDK (if installed)
- iOS Simulator availability
- Project configuration validity

## Example Output

```
Tano Doctor
-----------
[ok] Bun v1.1.0
[ok] Xcode 15.2
[ok] iOS Simulator available
[ok] tano.config.ts valid
[warn] Android Studio not found
```
